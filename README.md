# Init dotfiles

1. [initialize](https://www.chezmoi.io/user-guide/setup/) the dotfiles repo.

```bash
chezmoi init git@github.com:beanc904/dotfiles.git
```

2. check diff between repo and local and apply it.

```bash
chezmoi diff
chezmoi apply
```
