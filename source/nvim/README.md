# AstroNvimConfig

**NOTE:** This is for AstroNvim v6+

## 🛠️ Installation

### Make a backup of your current nvim and shared folder

```shell
mv ~/.config/nvim ~/.config/nvim.bkp
mv ~/.local/share/nvim ~/.local/share/nvim.bkp
mv ~/.local/state/nvim ~/.local/state/nvim.bkp
mv ~/.cache/nvim ~/.cache/nvim.bkp
```

### Clone it to your nvim config dir

```shell
git clone https://github.com/beanc904/AstroNvimConfig.git ~/.config/nvim
git clone git@github.com:beanc904/AstroNvimConfig.git ~/.config/nvim
```
